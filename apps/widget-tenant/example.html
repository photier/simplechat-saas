SimpleChat Widget Example

This page demonstrates how to embed the SimpleChat widget on your website.

Installation

Add this code before the closing </body> tag:

<script>
  window.simpleChatConfig = {
    chatId: 'your-chat-id',
    userId: 'guest-123',
    host: 'https://your-server.com',
    CustomData: {
      page: 'home',
      userAgent: navigator.userAgent
    },
    // Optional customizations
    mainColor: '#9F7AEA',
    titleOpen: "Let's chat!",
    titleClosed: 'Click to chat!',
    introMessage: 'Hello! How can we help you?',
    desktopHeight: 600,
    desktopWidth: 370
  };
</script>
<script src="https://your-cdn.com/simple-chat.min.js"></script>
Note: The widget will automatically appear on the bottom-right corner of your page. Replace the configuration values with your actual server details.

Configuration Options

  • chatId (required): Your unique chat identifier
  • userId (required): User identifier (e.g., guest-123)
  • host (required): Your chat server URL
  • CustomData (optional): Additional custom data
  • mainColor (optional): Widget theme color
  • titleOpen (optional): Title when chat is open
  • titleClosed (optional): Title when chat is closed
  • introMessage (optional): Initial greeting message
  • desktopHeight (optional): Widget height (default: 600px)
  • desktopWidth (optional): Widget width (default: 370px)
